I developed a modern Todo List web application using React, focusing on clean architecture and scalable state management. The project leverages React hooks such as useState, useEffect, useReducer, useContext, and useMemo to efficiently manage UI state and optimize performance. I
implemented a global state solution using the Context API combined with a reducer pattern (similar to Redux), ensuring a single source of truth and predictable state updates. The app also integrates localStorage for data persistence, allowing tasks to remain saved across page refreshes, along with reusable components and custom hooks for better code organization.

From a functionality perspective, the application fully supports CRUD operations (Create, Read, Update, Delete), enabling users to manage their daily tasks seamlessly. Features include adding new tasks, editing existing ones through modals, deleting tasks with confirmation dialogs, and toggling task completion status. Additional enhancements such as filtering (all, completed, pending), toast notifications for user feedback, and a responsive UI built with Tailwind CSS contribute to an improved user experience. This project demonstrates my ability to build maintainable, scalable front-end applications using modern React best practices.

قمت بتطوير تطبيق قائمة مهام (Todo List) حديث باستخدام React، مع التركيز على بناء هيكلية منظمة وقابلة للتوسع. اعتمدت في هذا المشروع على مجموعة من الـ Hooks مثل useState و useEffect و useReducer و useContext و useMemo لإدارة الحالة بكفاءة وتحسين الأداء. كما استخدمت Context API مع reducer لتطبيق نمط إدارة حالة مشابه لـ Redux، مما يضمن وجود مصدر بيانات موحد وتحديثات واضحة ومنظمة للحالة. بالإضافة إلى ذلك، تم استخدام localStorage لتخزين البيانات بشكل دائم، بحيث تبقى محفوظة حتى بعد إعادة تحميل الصفحة، مع تصميم مكونات قابلة لإعادة الاستخدام وتوظيف custom hooks لتحسين تنظيم الكود.

من ناحية الوظائف، يدعم التطبيق جميع عمليات CRUD (إنشاء، قراءة، تحديث، حذف)، حيث يمكن للمستخدم إضافة مهام جديدة، تعديلها من خلال نوافذ منبثقة، حذفها مع طلب تأكيد، بالإضافة إلى إمكانية تحديد حالة المهمة كمكتملة أو غير مكتملة. كما يوفر التطبيق ميزات إضافية مثل تصفية المهام (الكل، المنجزة، غير المنجزة)، وإشعارات تفاعلية (Toast) لتعزيز تجربة المستخدم، إلى جانب واجهة مستخدم متجاوبة تم تصميمها باستخدام Tailwind CSS. يعكس هذا المشروع قدرتي على تطوير تطبيقات واجهات أمامية احترافية وقابلة للتطوير وفق أفضل ممارسات React الحديثة
